# dotfiles

Config Files

## Installation

Clone this repo into your home directory:

```sh
cd ~
git clone git@github.com:codytwinton/dotfiles.git ~/.dotfiles
cd ~/.dotfiles

chmod +x run
./run install

chmod +x run_tests
./run_tests
```
